Guide For Git GUI

This application will allow a user to easily kick off the process of converting an existing project on their computer into a Git repo, and pushing the repo up to GitHub.



The following are requirements to use the application

  • External Library GitSubprocessClient
  • External Library GitHubApiClient
  • Java 8 or higher

Instructions

You will be prompted to enter your desired github url repo. Then following that you will have to enter the repo name,repo description,repo's visbilty,your user name and api access token.
